# Book Data Science Portfolio

A portfolio focused on books, cookbooks, publishing analytics, machine learning, NLP, and network science.

## Projects

### Book Influence Graph
Knowledge graph and network analysis of authors, books, genres, and influence.

### Bestseller Prediction
Machine learning model to predict book popularity and engagement.

### Recipe Knowledge Graph
Ingredient and recipe network analysis.

### Recipe Success Model
Predicting recipe engagement and ratings.

## Technology Stack

- Python
- SQLite / DuckDB
- NetworkX
- Sentence Transformers
- Power BI
- GitHub
